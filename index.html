About

I gathered the data by going through all CTF teams in the country stats of CTFtime for the years 2024 - 2026. As a consequence, a CTF team appears on this map, if

  • it is on CTFtime,
  • is associated with a country,
  • was among the top 1000 teams in 2024, 2025 or May 2026 and if
  • I was able to approximate the location

I determined the location by visiting the website linked on CTFtime. If the website didn't feature a location or if there was no website, I fell back to the affiliated university. If I fell back to the university, I tried to set the location near the cyber security or computer science department. As a consequence, the location is accurate if the marker is on a building and only a rough guess, if it is next to a building.

If you want to change the data, feel free to open a pull request on https://github.com/CyberTaskForce-Zero/ctf-map.
